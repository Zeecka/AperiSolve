# 快速入门

## 上传图片

前往[首页](/zh/)并拖入一张图片（PNG、JPG、JPEG、GIF、BMP、WebP 或 TIFF）。有两个可选设置会改变运行的内容：

- **密码** —— 会传递给支持密码的分析器（[steghide](/zh/wiki/tools/steghide)、openstego、jpseek、outguess）。没有候选口令时留空即可；此时各工具会以空密码运行，这在 CTF 中是常见的配置。
- **深度分析** —— 额外运行较慢的分析器，例如 outguess。

分析通常在几秒内完成；较大的图片或繁忙的队列可能需要更久。结果页面的 URL 是稳定且可分享的：以相同选项提交同一张图片会立即返回同一个页面。

## 解读结果

结果按分析器分组，每次的顺序都相同：

1. **位平面与颜色重映射**排在最前，因为它们是可视化结果：浏览生成的图片，寻找只出现在某个单独位平面中的轮廓、文字或二维码。
2. **文件与元数据工具**（`file`、`exiftool`、`identify`）能揭示格式不匹配、隐藏注释和编辑痕迹。
3. **文件雕复工具**（`binwalk`、`foremost`）列出嵌入在图片内部的文件；一旦有所发现，会出现一个下载按钮，提供包含提取文件的 `.7z` 压缩包。
4. **隐写提取器**（`zsteg`、`steghide`、`jpseek`、`jsteg`、`openstego`、`outguess`）尝试真正提取载荷。出现红色区块是正常现象：它只是表示该工具在给定密码下没有发现任何内容。

## 隐私与数据保留

上传的文件会被临时存储（默认 3 天）以便分享结果，随后自动删除。如果你误传了图片，只要它只从你自己的 IP 地址上传过，稍作等待后就可以在结果页面自行删除。

## 更进一步

通读[速查表](/zh/wiki/cheatsheet)，学习那些自动化工具无法覆盖的技巧；并阅读各工具页面，以便在本地对 Aperi'Solve 不支持的文件运行同样的分析。
