# Вики Aperi'Solve

Добро пожаловать в вики Aperi'Solve. Aperi'Solve — это бесплатная
онлайн-платформа, которая выполняет послойный анализ и обнаружение
стеганографии в изображениях: загрузите картинку на
[главной странице](/ru/), и все анализаторы запустятся автоматически.

Это вики описывает, как читать результаты и как работает каждый используемый
инструмент, чтобы вы могли воспроизвести и продолжить анализ на собственной
машине.

## С чего начать

- [Начало работы](/ru/wiki/getting-started) — как пользоваться Aperi'Solve и
  читать его результаты.
- [Шпаргалка по стеганографии для CTF](/ru/wiki/cheatsheet) — практический
  чек-лист для заданий на изображения, аудио и форматы файлов.

## Руководства по инструментам

У каждого анализатора, который обрабатывает вашу загрузку, есть своя
страница: что делает инструмент, какую именно команду выполняет Aperi'Solve,
как интерпретировать вывод и как установить его локально.

- [Декомпозиция битовых плоскостей](/ru/wiki/tools/decomposer) —
  визуализация каждого бита каждого цветового канала.
- [Преобразование цветов](/ru/wiki/tools/color_remapping) — выявление
  скрытых данных с помощью преобразований палитры.
- [zsteg](/ru/wiki/tools/zsteg) — обнаружение LSB-стеганографии в PNG и BMP.
- [steghide](/ru/wiki/tools/steghide) — извлечение данных, скрытых в
  JPEG/BMP с парольной фразой.
- [binwalk](/ru/wiki/tools/binwalk) — поиск и извлечение файлов, встроенных
  в изображение.
- [foremost](/ru/wiki/tools/foremost) — вырезание встроенных файлов из
  изображения.
- [exiftool](/ru/wiki/tools/exiftool) — чтение метаданных (EXIF, XMP,
  IPTC...).
- [pngcheck](/ru/wiki/tools/pngcheck) — проверка структуры PNG и поиск
  повреждённых чанков.
- [PCRT](/ru/wiki/tools/pcrt) — обнаружение и восстановление повреждённых
  PNG-файлов.
- [OutGuess](/ru/wiki/tools/outguess) — извлечение скрытых данных из
  изображений JPEG.
- [jsteg](/ru/wiki/tools/jsteg) — извлечение LSB-данных из
  DCT-коэффициентов JPEG.
- [JPHide/JPSeek](/ru/wiki/tools/jpseek) — извлечение полезной нагрузки
  JPHide из JPEG.
- [OpenStego](/ru/wiki/tools/openstego) — извлечение рандомизированной
  LSB-стеганографии.
- [identify](/ru/wiki/tools/identify) — просмотр свойств изображения с
  помощью ImageMagick.
- [file](/ru/wiki/tools/file) — определение настоящего типа загруженного
  файла.
- [strings](/ru/wiki/tools/strings) — поиск читаемого текста внутри файлов
  изображений.

Ваш вклад приветствуется на

## Что такое стегоанализ?

Стеганография прячет данные внутри безобидного файла-контейнера — для
изображений это означает манипуляции с битами пикселей, элементами палитры,
полями метаданных или самой структурой файла. Стегоанализ — это практика
обнаружения таких скрытых данных. Ни один инструмент не покрывает все
техники, поэтому Aperi'Solve запускает целую батарею анализаторов
параллельно и позволяет сравнить их результаты с одного взгляда.
